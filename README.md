# 🎧Spotify Playlist Dashboard

Dashboard interactivo para analizar y visualizar las métricas de una playlist de Spotify. El proyecto permite explorar desde la cronología de lanzamientos hasta detalles de duración y contenido de la biblioteca musical de forma visual y elegante.

## 🚀 Características
- **Métricas Clave (KPIs):** Conteo total de canciones, artistas únicos, álbumes y duración media.
- **Identidad Musical:** Análisis de los artistas y álbumes con mayor presencia en la lista.
- **Línea del Tiempo:** Visualización de lanzamientos por año y comparativa de catálogo (Actual vs. Clásico).
- **Análisis de Duración:** Top de canciones más extensas y breves con estadísticas de variabilidad.
- **Contenido Explícito:** Gráfico de distribución de contenido explícito con diseño personalizado y visualización de impacto.
- **Explorador de Datos:** Tabla detallada con portadas de álbumes y enlaces directos para escuchar en Spotify.

## 🛠️ Tecnologías Utilizadas
* [Python](https://www.python.org/) - Lenguaje de programación.
* [Streamlit](https://streamlit.io/) - Framework para la creación de la interfaz web.
* [Plotly](https://plotly.com/python/) - Gráficos interactivos y dinámicos.
* [Pandas](https://pandas.pydata.org/) - Procesamiento y manipulación de datos.
* [Spotipy](https://spotipy.readthedocs.io/) - Librería cliente para la API de Spotify.
